## Thesis

The thesis provides a comprehensive description of the project, including:

- Background and literature review
- System design and hardware development
- Embedded software implementation
- Python data acquisition and processing tools
- Experimental methodology
- Calibration procedures
- Results and discussion
- Conclusions and recommendations for future work

The source code, embedded firmware, and engineering drawings contained in this repository were developed as part of the work presented in the accompanying thesis.

## Engineering Drawings

The included engineering drawings document the design and manufacture of the experimental components, including:

- Accelerometer mounting hardware
- Lighthouse geometry
- Base and upper sections for:
  - Fastnet
  - Skerryvore
  - Ar Men
  - Wolf Rock

These drawings provide the dimensions and construction details required to reproduce the physical sections used during experimental validation.
